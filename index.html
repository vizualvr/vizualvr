About Vizual VR

Vizual VR is a web application for visualizing Excel charts in 3D space using Three.JS, with the support for the experimental WebVR API on Firefox Nightly and select Chromium builds.

Get a sense of scale like never before to internalize and remember important data. Move around between columns and experience how things feel when you're alongside your information.

Meet the Developer
